## 项目概述
这是一个基于HTML/CSS/JavaScript构建的个人照片画廊网站，名为「阿彤的时光画廊」，以艺术画廊为设计风格，展示照片时光轴和照片墙。

## 核心功能
1. **画廊首页**：展示画廊标题和进入按钮
2. **时光轴**：横向滚动展示按文件夹分类的照片胶囊
3. **照片墙**：展示所有照片的网格布局
4. **关于页面**：网站介绍

## 目录结构
```
/照片/
├── index.html          # 首页
├── timeline.html       # 时光轴页面
├── gallery.html        # 照片墙页面
├── about.html          # 关于页面
├── script.js           # 主要交互逻辑
├── styles.css          # 样式文件
├── generate_timeline.py # 生成时光轴HTML的脚本
├── update_timeline.py   # 更新时光轴数据的脚本
└── 时光画廊/             # 照片存储目录
    ├── 01_合照/         # 分类目录
    ├── 02_生日照/
    ├── 03_日常漂亮饭/
    ├── 04_云南旅游/
    ├── 05_日常美照/
    ├── 06_日常自拍/
    └── 07_毕业照/
```

## 核心代码架构
- **页面结构**：使用HTML构建页面框架，导航栏通用，内容区域按页面功能划分
- **样式系统**：CSS变量定义配色方案，响应式设计适配不同设备
- **交互逻辑**：JavaScript实现时光轴滚动、照片加载、动画效果等
- **数据生成**：Python脚本自动从照片目录生成时光轴HTML结构

## 关键技术点
1. **时光轴实现**：使用横向滚动布局展示照片胶囊，每个胶囊对应一个分类目录
2. **照片加载**：采用懒加载技术提升性能
3. **动画效果**：CSS过渡和JavaScript控制的淡入、缩放等动画
4. **自动生成**：Python脚本自动遍历照片目录生成时光轴HTML代码

## 运行方式
- 直接在浏览器中打开HTML文件即可运行
- 无需构建或编译步骤
- 照片和HTML文件需在同一目录结构下

## 开发流程
1. 将照片按分类放入「时光轴」目录下
2. 运行generate_timeline.py或update_timeline.py脚本更新时光轴
3. 在浏览器中查看页面效果

## 注意事项
- 照片文件名应使用中文或英文，避免特殊字符
- 目录结构变更后需重新运行Python生成脚本
- JavaScript依赖DOM结构，修改HTML后需同步更新script.js逻辑
